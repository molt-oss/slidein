Privacy Policy

Last updated: March 30, 2026

1. Introduction

slidein is an open-source, self-hosted Instagram DM automation tool. Each user deploys their own instance on their own Cloudflare account. The slidein project does not operate any central server and does not collect, store, or process any user data.

2. Data Controller

When you deploy slidein, you are the data controller. All data is stored in your own Cloudflare D1 database, under your Cloudflare account. The slidein project and its maintainers have no access to your instance or data.

3. Data We Process

A self-hosted slidein instance may process the following data through the Instagram/Meta API:

  • Instagram User IDs — to identify conversation participants
  • Message Content — incoming DM text to match keyword rules and trigger automated responses
  • Comment Content — public post comments to trigger DM responses (if configured)
  • Contact Records — Instagram user IDs with optional tags/scores for audience management

4. How Data Is Used

Data processed by your slidein instance is used solely for:

  • Matching incoming messages against your keyword rules
  • Sending automated DM responses via the Instagram API
  • Managing contacts and broadcast audiences
  • Running scenario-based message sequences
  • AI-powered auto-replies (if you configure an AI API key)

No data is sold, shared with third parties, or used for advertising purposes.

5. Data Storage & Security

  • All data is stored in your Cloudflare D1 database, encrypted at rest by Cloudflare.
  • API secrets (Meta tokens, app secrets) are stored as Cloudflare Worker Secrets, not in plain text.
  • The dashboard is protected by password authentication.
  • Webhook payloads are verified using HMAC-SHA256 signature validation.

6. Data Retention

Data retention is controlled entirely by you. You can delete contacts, conversation data, and keyword rules at any time through the dashboard. Deleting your Cloudflare D1 database removes all stored data permanently.

7. Third-Party Services

Your slidein instance connects to:

  • Meta/Instagram API — to receive and send messages (required)
  • Cloudflare — hosting infrastructure (required)
  • AI providers (OpenAI, Anthropic, etc.) — only if you configure AI auto-reply (optional)

Each third-party service is governed by its own privacy policy.

8. User Rights

Since slidein is self-hosted, data subject requests (access, deletion, portability) should be directed to the operator of each slidein instance. As the operator, you are responsible for complying with applicable data protection laws (GDPR, CCPA, etc.).

9. Changes to This Policy

This privacy policy may be updated from time to time. Changes will be reflected in the "Last updated" date above and committed to the GitHub repository.

10. Contact

For questions about this privacy policy or the slidein project, please open an issue on GitHub.
