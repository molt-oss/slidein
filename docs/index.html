Prerequisites

Before you start, make sure you have the following:

📸

Instagram Business or Creator Account

Personal accounts won't work with the API. Switch in Instagram Settings → Account → Switch to Professional Account → choose Business or Creator.

👤

Facebook Account

Required for the Meta Developer Portal. facebook.com

☁️

Cloudflare Account (free)

This is the cloud server that runs your slidein. No credit card required — the free plan covers everything. Sign up →

📦 Workers = a tiny server that runs your code (100k requests/day free)

🗄️ D1 = a database for your contacts & rules (5 GB free)

🌐 Pages = hosting for your dashboard (unlimited free)

→ Enough for ~5,000 followers at $0/month ✨

🟢

Node.js 20+ & pnpm

nodejs.org · Install pnpm: npm install -g pnpm

1

Clone & Install

bash 
git clone https://github.com/molt-oss/slidein.git
cd slidein
pnpm install
2

Meta App Setup

2.1 — Register as a Developer

Go to developers.facebook.com and register your Facebook account as a developer if you haven't already.

2.2 — Create a New App

  1. Go to developers.facebook.com/apps
  2. Click Create App
  3. Fill in app name (e.g., slidein) and contact email → Next

2.3 — Choose Use Case

You'll see a use case selection screen. This is the important part:

  1. In the left filter, click "Business Messaging"
  2. Check "Manage messages and content on Instagram" ✅
  3. Click Next
  4. Select your Business Portfolio (Facebook Page) — create one if you don't have it
  5. Click Create App
⚠️ Don't select "Other" or "Advertising" — Instagram API won't be available! Make sure to pick "Business Messaging" → "Manage messages and content on Instagram".
⚠️ The dashboard will show "Business Verification" and "App Review" steps — you can skip both! slidein works in Development Mode, which doesn't require either.

2.4 — Add Instagram Tester

  1. Go to App Roles → Roles in the left sidebar
  2. Under Instagram Testers, click Add Instagram Testers
  3. Enter your Instagram username
  4. On your phone, open Instagram → Settings → Website Permissions → Tester Invitations → Accept

2.5 — Get Your Credentials

You'll need these values in the next step. Keep them handy!

Credential Where to find it
App Secret App Settings → Basic → App Secret (click Show)
Access Token Instagram → API Setup → Generate Token (select your IG account)
Instagram Account ID Instagram → API Setup → shown as "Instagram Account ID"
3

One-Command Setup ⚡

This single command handles everything — database creation, migrations, secrets, deployment, and dashboard configuration:

bash 
pnpm run setup

🪄 The setup wizard will interactively guide you through everything. Just paste the values from Step 2 when prompted. Here's what it does automatically:

✓ Checks Cloudflare login (opens browser if needed)
✓ Asks for your Meta credentials interactively
✓ Creates D1 database & writes the ID to wrangler.toml
✓ Runs all 6 database migrations
✓ Sets all Worker secrets
✓ Deploys the Worker
✓ Generates a dashboard admin password

🎉 When complete, the setup will display your Worker URL and dashboard admin password. Save these — you'll need them in the next steps!

4

Connect Webhook

  1. Go to your Meta app → Webhooks in the left sidebar
  2. Select Instagram from the dropdown
  3. Click Subscribe to this object (or Edit if already set up)

  4. Fill in:

    Callback URL: https://slidein-worker.YOUR_SUBDOMAIN.workers.dev/webhook
    Verify Token: your META_VERIFY_TOKEN value
  5. Click Verify and Save
  6. Subscribe to these fields:
    messages messaging_postbacks comments
5

Test It! 🎉

1

Open the dashboard and create a keyword rule:

Keyword: hello → Response: Hey! Thanks for reaching out 👋

2

Open Instagram and send a DM with "hello" to your own business account (from another account or use the API test tool).

3

If you get an automatic reply — you're all set! 🚀

Troubleshooting

Problem Solution
Webhook verification fails Make sure your META_VERIFY_TOKEN matches exactly in both the worker secret and Meta console.
No DM reply received Check that you've subscribed to messages in the webhook fields. Also verify your Instagram account is a business/creator account.
Setup script fails Make sure you're logged into Cloudflare (npx wrangler login) and have Node.js 20+ installed. You can safely re-run pnpm run setup — it picks up where it left off.
Dashboard shows "Unauthorized" Use the admin password shown at the end of pnpm run setup. If you lost it, check your .env.local file or re-run setup.
Token expired / invalid Instagram access tokens can expire. Re-generate from Meta Developer Console → Instagram → API Setup.
wrangler not found Use npx wrangler (included in dev dependencies) or install globally: npm i -g wrangler
